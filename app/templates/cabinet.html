📖 Инструкции по подключению

Hiddify (рекомендуется)

  1. Установите Hiddify из App Store
  2. Откройте приложение → нажмите + (добавить профиль)
  3. Выберите «Добавить по ссылке»
  4. Вставьте ссылку подписки (скопируйте выше) или отсканируйте QR-код
  5. Нажмите «Сохранить», затем включите подключение

Streisand

  1. Установите Streisand из App Store
  2. Нажмите + → «Импорт из буфера»
  3. Скопируйте ссылку подписки и вставьте
  4. Подтвердите добавление → подключайтесь

Hiddify (рекомендуется)

  1. Установите Hiddify из Google Play или GitHub
  2. Откройте → нажмите «Новый профиль»
  3. Выберите «Добавить по ссылке» и вставьте ссылку подписки
  4. Или нажмите сканер QR → наведите на QR-код выше
  5. Включите VPN-подключение

v2rayNG

  1. Установите v2rayNG из Google Play
  2. Меню ⋮ → «Импорт из буфера»
  3. Или: ⋮ → «Подписки» → добавьте URL подписки
  4. Выберите сервер → нажмите ▶️ для подключения

Hiddify (рекомендуется)

  1. Скачайте Hiddify с github.com/hiddify
  2. Установите и запустите
  3. Нажмите «Новый профиль» → «По ссылке»
  4. Вставьте ссылку подписки → «Сохранить»
  5. Выберите сервер и включите

Nekoray

  1. Скачайте Nekoray с GitHub
  2. Распакуйте и запустите nekoray.exe
  3. Сервер → «Добавить ссылку из буфера»
  4. Или: Настройки → «Группы» → подписка по URL
  5. ПКМ на сервер → «Запустить»

Hiddify

  1. Скачайте Hiddify из GitHub (macOS .dmg)
  2. Установите и откройте
  3. Добавьте профиль по ссылке подписки
  4. Выберите сервер → подключайтесь
