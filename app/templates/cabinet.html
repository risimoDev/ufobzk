📖 Инструкции по подключению

Hiddify (рекомендуется)

  1. Установите Hiddify из App Store
  2. Нажмите + → «Добавить по ссылке»
  3. Вставьте ссылку подписки или отсканируйте QR-код
  4. Появятся оба сервера (🇷🇺 Россия + 🇳🇱 Европа)
  5. Включите подключение — маршрутизация автоматическая

Hiddify / v2rayNG

  1. Установите Hiddify из Google Play / GitHub
  2. Нажмите «Новый профиль» → «По ссылке»
  3. Вставьте ссылку подписки
  4. Или используйте v2rayNG → Подписки → добавьте URL
  5. Включите VPN

Hiddify / Nekoray

  1. Скачайте Hiddify с github.com/hiddify
  2. «Новый профиль» → «По ссылке»
  3. Вставьте ссылку подписки → «Сохранить»
  4. Выберите сервер и подключайтесь

Hiddify

  1. Скачайте Hiddify (.dmg) с GitHub
  2. Добавьте профиль по ссылке подписки
  3. Подключайтесь
